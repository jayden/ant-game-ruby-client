# Basic Ant Game Client in Ruby

## Prerequisites

You will need Ruby and Rspec (if you want to run the tests) installed.

## Testing

Ensure the app is running by running the tests:

```bash
rspec
```

## Running

To start the app, run within the root directory:

```bash
ruby lib/app.rb
```
